如何激活Windows10系统
发表于 2020-07-22| 更新于 2020-07-22
字数总计:703|阅读时长: 2 分钟
|阅读量:

近来许多小伙伴问我关于Windows10系统的激活问题。我可能不能一个个详细的解答或者一个个帮忙激活。为了避免小伙伴再去淘宝买激活密钥，下面我将写一篇简单的激活操作教程，希望能帮助到各位小伙伴。

如何判断系统是否激活

  • （简单）屏幕右下角出现 “激活Windows” 则系统处于未激活或者激活过期状态。
  • （进阶）在Powershell中输入slmgr.vbs -xpr，根据提示判断是否激活状态。

为什么我们需要激活

  • 为了好看
  • 为了安全
  • 你说了算

如何激活

  • 下载激活工具点我跳转 (1.7MB)
  • 解压激活工具
  • 按下Windows键输入PowerShell，右键以管理员权限运行。
  • 在PowerShell界面中输入 slmgr.vbs /upk卸载当前激活密钥。
  • 右键以管理员权限运行解压后的激活工具中的 Activation.cmd
  • 按任意键进入第二界面，再按1选择激活当前系统
  • 等待完成，大概需要几分钟。
  • 在Powershell中输入slmgr.vbs -xpr，若提示已永久激活则激活成功。

激活过程中出错了怎么办

首先，在不知道什么问题的情况下，建议按照步骤重新做一遍。

  • 网络问题

    找一个稳定的网络接入点，重复操作

  • 各项操作中提示

    • 错误:0xC004F025 拒绝访问
    • 所请求的操作需要提升特权

    这是因为你没有以管理员权限运行导致的

    请右键以管理员权限运行

  • 激活工具提示 0x80070422 服务未启动

    这是由于 Windows update 服务未启动导致的。

    1. Windows 键 + R 键输入 services.msc并回车
    2. 在services界面内找到 Windows Update服务
    3. 调整Windows Update服务为手动，并启动服务
    4. 重新以管理员权限打开激活工具，重新激活

  • 少数品牌的笔记本电脑预装系统锁定权限

    如果上面的问题你都没出现，但仍然未能激活。

    如果上面的问题你都出现了，并且按照我上方提示仍未能激活。

    这便可能是预装系统的问题，建议直接联系我，我飞过去帮你操作（重装解决一切问题）。

  • 尚未发现其他问题

一些小提示

  • Windows键是一个微软标志的按键，部分品牌笔记本电脑上是Start键

  • 各项命令行操作都要以管理员权限运行

  • 按下任意键意思是按下键盘上任何一个按键都可以的意思

  • 小朋友，你是不是还有很多疑惑？

    要是上面仍然没有解决你的问题，来找我吧~我看到消息就会找你的。
